Task 1: IMAGE GALLERY
A sleek, responsive, and functional image gallery built with HTML5, CSS3, and JavaScript. This project features category filtering, a smooth lightbox viewer with navigation, and a fully responsive grid layout.
##Features
Category Filtering: Sort images by type (Nature, Pizza, Burgers, etc.) using dynamic filters.
Lightbox View: Click on any image to open it in a full-screen modal.
Next/Prev Navigation: Seamlessly browse through images within the lightbox.
Responsive Design: Optimized for desktop, tablet, and mobile screens using CSS Grid.
Smooth Animations: Includes hover effects and transitions for a polished user experience.
##Technologies Used
HTML5: Semantic structure.
CSS3: Custom styling, Flexbox, and Grid.
JavaScript (ES6): DOM manipulation, filtering logic, and lightbox functionality.

Task 2: Caclculator
A sleek, responsive calculator web application built using HTML5, CSS3, and Vanilla JavaScript. This project demonstrates core frontend development skills including DOM manipulation, event handling, and CSS Grid layouts.

###Features
Full Arithmetic Suite: Supports Addition, Subtraction, Multiplication, and Division.
Real-time Display: Instant updates as you input numbers and operators.
Keyboard Support: Fully functional using a physical keyboard (Numbers, Enter, Backspace, and Operators).
Glassmorphism UI: A modern, frosted-glass design with responsive button hover effects.
Error Handling: Prevents crashes from invalid mathematical expressions.
Clear & Delete: Functions to clear the entire screen (AC) or delete the last character entered (DEL).

##Tech Stack
HTML5: Semantic structure for the calculator interface.
CSS3: Custom styling featuring CSS Grid, Flexbox, and backdrop-filters for the glass effect.
JavaScript (ES6): Logic for mathematical evaluations and keyboard event listeners.
