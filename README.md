PROJECT 1: IMAGE GALLERY
A sleek, responsive, and functional image gallery built with HTML5, CSS3, and JavaScript. This project features category filtering, a smooth lightbox viewer with navigation, and a fully responsive grid layout.
##Features
Category Filtering: Sort images by type (Nature, Pizza, Burgers, etc.) using dynamic filters.
Lightbox View: Click on any image to open it in a full-screen modal.
Next/Prev Navigation: Seamlessly browse through images within the lightbox.
Responsive Design: Optimized for desktop, tablet, and mobile screens using CSS Grid.
Smooth Animations: Includes hover effects and transitions for a polished user experience.
##Technologies Used
HTML5: Semantic structure.
CSS3: Custom styling, Flexbox, and Grid.
JavaScript (ES6): DOM manipulation, filtering logic, and lightbox functionality.
